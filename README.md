# Sistema de Gestión de Préstamos de Herramientas🔧📝

## ⚡️Descripción

✨Este proyecto es un sistema de gestión de préstamos de herramientas desarrollado en Python. Permite a los administradores gestionar usuarios y herramientas, mientras que los usuarios pueden solicitar préstamos de herramientas disponibles. El sistema incluye funcionalidades de autenticación básica, registro de historial de acciones y validaciones de entrada.✨

## 🔥Características

- **👥Gestión de Usuarios**: Agregar, actualizar, eliminar y listar usuarios.
- **🔧Gestión de Herramientas**: Agregar, actualizar, eliminar y listar herramientas.
- **🦖Préstamos**: Los usuarios pueden solicitar préstamos de herramientas.
- **📝Historial**: Registro de todas las acciones realizadas en el sistema.
- **🧑‍💻Autenticación**: Acceso diferenciado para administradores y usuarios regulares.
- **✅Validaciones**: Entradas validadas para asegurar la integridad de los datos.

## 💡Requisitos

- Python 3.x

## 🌐Instalación y Ejecución

1. Clona o descarga el proyecto.
2. Asegúrate de tener Python 3.x instalado.
3. Ejecuta el archivo principal:

   ```
   python main.py
   ```

4. Sigue las instrucciones en pantalla para acceder como administrador o usuario.

## 🏗️Estructura del Proyecto

- `main.py`: Punto de entrada del programa.
- `menu.py`: Contiene los menús y lógica de navegación.
- `gestionUsuarios.py`: Funciones para gestionar usuarios.
- `gestionHerramientas.py`: Funciones para gestionar herramientas.
- `gestionPrestamos.py`: Funciones para gestionar préstamos.
- `gestionarJson.py`: Utilidades para manejar archivos JSON.
- `validaciones.py`: Funciones de validación de entrada.
- `*.json`: Archivos de datos (Usuarios.json, Herramientas.json, etc.).

## 🔍️Uso

- **Administrador**: Ingresa la contraseña para acceder al menú de administración, donde puedes gestionar usuarios, herramientas, préstamos y ver historiales.
- **Usuario**: Selecciona la opción de usuario para listar información y solicitar préstamos.
